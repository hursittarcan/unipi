Common Data Structures

Python provides several built-in data structures that are essential for programming.

Lists

# Creating a list
fruits = ["apple", "banana", "cherry"]

# Accessing elements
print(fruits[0])  # Output: apple

# Adding elements
fruits.append("orange")

# List length
print(len(fruits))  # Output: 4

Lists are ordered, mutable collections that can contain elements of different types.

  • Access elements with index: list[index]
  • Add elements with append() or insert()
  • Remove elements with remove() or pop()

Dictionaries

# Creating a dictionary
student = {
  "name": "John",
  "age": 20,
  "courses": ["Math", "CS"]
}

# Accessing values
print(student["name"])  # Output: John

# Adding new key-value pairs
student["grade"] = "A"

Dictionaries store key-value pairs and provide fast lookups by key.

  • Access values with key: dict[key]
  • Check if key exists: key in dict
  • Get all keys: dict.keys()

Tuples

# Creating a tuple
point = (10, 20)

# Accessing elements
x, y = point
print(x)  # Output: 10

# Tuples are immutable
# This will cause an error:
# point[0] = 15

Tuples are immutable ordered collections, often used for fixed data.

  • Cannot be modified after creation
  • Faster than lists for iteration
  • Can be used as dictionary keys
