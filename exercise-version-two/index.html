Logical Errors in Selection Structures

Introduction

A logical error is a mistake in a program’s logic that causes incorrect behavior or output, even though the code runs without crashing. These errors are common in selection structures like if, else if, and else.

Exercise 1: Grading System


let score = 75;

if (score > 90)
  console.log("Grade: A");
else if (score > 80)
  console.log("Grade: B");
else if (score > 70)
  console.log("Grade: C");
else
  console.log("Grade: F");

What will this code output?

Exercise 2: Login Logic


let username = "admin";
let password = "1234";

if (username === "admin")
  if (password === "admin")
    console.log("Login successful");
  else
    console.log("Invalid username");

Summary

  • Logical errors are tricky because they don’t crash your code, but still produce incorrect results.
  • Carefully order your conditions in selection structures.
  • Always test different input cases to catch logical mistakes.
