Logic Flow Evaluator

This tool helps you understand how different input values affect selection structures with logical errors.

Problem 1 – Wrong Order of Conditions 
if (num < 20)
  result = "Under 20";
else if (num < 10)
  result = "Under 10";
else
  result = "20 or more";
Problem 2 – Assignment Instead of Comparison 
if (mode = "test")
  result = "Test Mode Enabled";
else
  result = "Normal Mode";
Problem 3 – Missing Else 
if (grade >= 90)
  result = "A";
if (grade >= 80)
  result = "B";
else
  result = "C";
